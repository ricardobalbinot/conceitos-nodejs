## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/ricardobalbinot/conceitos-nodejs.git

# Go into the repository
$ cd conceitos-nodejs

# Install dependencies
$ yarn

# Run server at port 3333
$ yarn dev

# Run CRUD tests
$ yarn test
```
